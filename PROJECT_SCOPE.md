# 128GB 车规级 UFS 3.1 项目范围

## 项目概述
开发符合车规级标准的 128GB UFS 3.1 存储解决方案，满足汽车电子领域高可靠性、宽温运行、长时间稳定工作的需求。

## 核心规格
| 项 | 规格 |
|-----|-----|
| 容量 | 128GB |
| 标准 | UFS 3.1 |
| 等级 | 车规级（AEC-Q100） |
| 工作温度 | -40°C ~ +105°C |
| 顺序读取 | ≥ 2100 MB/s |
| 顺序写入 | ≥ 1600 MB/s |
| 随机读取 | ≥ 400K IOPS |
| 随机写入 | ≥ 300K IOPS |
| 平均无故障时间 | ≥ 200万小时 |

## 应用场景
- 智能座舱系统
- 自动驾驶存储单元
- 车载信息娱乐系统
- 车规级数据记录仪
- 域控制器存储单元
