## ⚠️ 关键注意事项

### D1-EM (电迁移) 关键点
```
🔴 电流密度必须精确控制 (±1%)
🔴 温度必须稳定 (±2°C)
🔴 电阻测量必须使用 Kelvin 结构
🟡 注意 Joule 加热导致的自升温
```

### D2-TDDB (介质击穿) 关键点
```
🔴 高压操作安全 (双人作业)
🔴 泄漏电流测量精度 (pA 级)
🔴 氧化层电场计算准确
🟡 注意应力模式选择 (CVS vs CCS)
```

### D3-HCI (热载流子) 关键点
```
🔴 最坏情况温度 (通常 25°C 而非高温)
🔴 衬底电流最大化条件
🔴 快速测量减少恢复
🟡 注意三种应力模式的选择
```

### D4-BTI (偏压温度) 关键点 ⭐ **红框重点**
```
🔴 快速测量 (<1 秒) 避免恢复效应
🔴 NBTI 比 PBTI 更严重 (优先关注 PMOS)
🔴 应力 - 测量切换时间 <100ms
🟡 完全表征会导致部分恢复 (数据标注)
```

### D5-SM (应力迁移) 关键点
```
🔴 关注通孔和拐角等应力集中区域
🔴 电阻变化是主要判定指标
🔴 X-Ray/SAT用于失效定位
🟡 注意与 EM 失效的区分
```
