# AEC-Q100 D4 BTI 测试执行指导书

**文档编号**: AEC-D4-BTI-001  
**版本**: 1.0  
**创建日期**: 2026-03-26  
**依据标准**: AEC-Q100-004, JEDEC JESD47  
**适用产品**: UFS 3.1 车规级存储控制器  
**测试等级**: Grade 1 (-40°C ~ +125°C)

---

## 📋 1. 测试目的

评估 MOSFET 在偏压和温度共同应力下的抗 BTI 效应能力，确保在 10 年使用寿命内阈值电压漂移在可接受范围内。

### BTI 失效机理
```
高温 + 栅极偏压 → Si-H 键断裂 → 界面态生成 (Dit 增加)
               → 阈值电压漂移 (ΔVth)
               → 跨导下降 (ΔGm)
               → 电路延迟增加 → 时序违例 → 系统失效
```

---

## 🔬 2. 测试原理

### 两种 BTI 类型

| 类型 | NBTI (Negative BTI) | PBTI (Positive BTI) |
|------|---------------------|---------------------|
| **影响器件** | PMOS | NMOS |
| **偏压条件** | Vg < 0 (负偏压) | Vg > 0 (正偏压) |
| **严重性** | 🔴 更严重 (n 指数 0.2-0.3) | 🟡 较轻 (n 指数 0.15-0.2) |
| **恢复效应** | 显著 (部分可恢复) | 较小 |
| **激活能** | 0.1-0.2 eV | 0.1-0.15 eV |

### 反应 - 扩散模型 (R-D Model)
```
ΔVth = A × (Vg - Vth0)^n × t^m × exp(-Ea/kT)

其中:
- ΔVth = 阈值电压漂移
- Vg = 栅极电压
- Vth0 = 初始阈值电压
- n = 电压指数 (典型 1-2)
- m = 时间指数 (典型 0.15-0.3)
- t = 应力时间
- Ea = 激活能 (NBTI: 0.1-0.2 eV)
- T = 绝对温度
```

### 恢复效应
```
应力阶段: Si-H 键断裂 → 界面态增加 → Vth 漂移
           ↓
弛豫阶段: H 原子回填 → 部分界面态修复 → Vth 部分恢复

恢复比例：NBTI 约 30-50%, PBTI 约 10-20%

注意：测试测量需考虑恢复效应，采用快速测量技术
```

## 📝 5. 测试步骤

### 5.1 测试前准备

#### 步骤 1: Test Key 设计
```
□ 包含 PMOS 和 NMOS 晶体管
□ 包含不同尺寸 (W/L) 器件
□ 包含核心和 IO 器件
□ Kelvin 连接用于精确测量
□ 设计审核通过
```

#### 步骤 2: 样品准备
```
□ 选取 77 个样品 (同一批次)
□ 编号：BTI-P-001 ~ BTI-P-077 (PMOS)
       BTI-N-001 ~ BTI-N-077 (NMOS)
□ 初始电性测试 (Vth, Idsat, Gm, S)
□ 记录初始数据
```

#### 步骤 3: 设备准备
```
□ 老化炉预热至 125°C
□ 校准电压源和测量单元
□ 配置快速测量程序
□ 切换时间验证 (< 1s)
```

### 5.2 初始测量 (T=0h)

#### 步骤 4: 转移特性测量
```
PMOS:
1. Vd = -Vdd, Vs = 0, Vb = Vdd
2. 扫描 Vg: Vdd → 0
3. 测量 Id
4. 提取 Vth0, Gm0, S0

NMOS:
1. Vd = Vdd, Vs = 0, Vb = 0
2. 扫描 Vg: 0 → Vdd
3. 测量 Id
4. 提取 Vth0, Gm0, S0
```

#### 步骤 5: 记录初始参数
```
| 样本 ID | Vth0 (V) | Idsat0 (mA) | Gm0 (mS) | S0 (mV/dec) |
|---------|----------|-------------|----------|-------------|
| BTI-P-001 | -0.45 | 1.25 | 2.8 | 85 |
| BTI-N-001 | +0.42 | 1.30 | 2.9 | 82 |
| ... | ... | ... | ... | ... |
```

### 5.3 施加应力

#### 步骤 6: 安装样品
```
1. 佩戴防静电手环和防热手套
2. 将样品安装到测试夹具
3. 连接电压源和测量单元
4. 放入老化炉
5. 关闭炉门，升温至 125°C
```

#### 步骤 7: 施加栅压
```
PMOS (NBTI):
- Vg = -1.8V (负偏压)
- Vs = Vd = 0V (接地)
- Vb = Vdd (衬底偏压)

NMOS (PBTI):
- Vg = +1.8V (正偏压)
- Vs = Vd = 0V (接地)
- Vb = 0V (衬底接地)

设置自动监控程序
```

### 5.4 过程监控

#### 步骤 8: 定时测量（关键步骤）

```
测量时间点：0, 1, 10, 100, 500, 1000 小时

每个时间点的操作 (必须在 <1 秒内完成):

1. 暂停应力 (打开 Vg 开关)
   ↓ (t < 100ms)
2. 快速测量 Vth
   - 使用快速扫描或单点测量
   - 记录 Vth_stress
   ↓ (t < 500ms)
3. 恢复应力 (关闭 Vg 开关)
   ↓
4. 计算 ΔVth = Vth_stress - Vth0
5. 判定：|ΔVth| > 50mV → 失效预警
6. 记录数据

注意：整个过程必须在 <1 秒内完成，以避免恢复效应影响测量结果
```

#### 步骤 9: 完全表征（仅在关键时间点）

```
在 0, 100, 500, 1000 小时进行完整表征:

1. 暂停应力
2. 冷却至 25°C (自然冷却或强制风冷)
3. 测量完整 Id-Vg 曲线
4. 提取 Vth, Idsat, Gm, S
5. 测量 Id-Vd 曲线
6. 记录所有参数
7. 恢复应力

注意：完全表征会导致部分恢复，数据需标注"弛豫后"
```

### 5.5 测试结束

#### 步骤 10: 最终测量
```
1. 关闭应力电压
2. 冷却至 25°C
3. 执行完整电性表征
4. 记录最终数据
5. 计算总退化量
```

## ✅ 7. 判定标准

### Pass/Fail 判定

| 判定 | 标准 |
|------|------|
| **PASS** | 77 个样品中 0 个失效 (快速测量：\|ΔVth\| ≤ 50mV @ 1000h) |
| **FAIL** | 77 个样品中 ≥1 个失效 (\|ΔVth\| > 50mV) |

### 退化参数要求

| 参数 | 符号 | 要求 (1000h) | 单位 |
|------|------|-------------|------|
| 阈值电压漂移 (快速) | ΔVth_fast | ≤ 50 | mV |
| 阈值电压漂移 (弛豫) | ΔVth_relax | ≤ 40 | mV |
| 饱和电流变化 | ΔIdsat | ≤ 10 | % |
| 跨导变化 | ΔGm | ≤ 10 | % |
| 亚阈值斜率变化 | ΔS | ≤ 20 | % |

### 寿命外推要求

```
基于测试数据外推使用条件下的寿命:

使用条件：125°C, Vdd = 1.5V
要求：ΔVth ≤ 50mV @ 10 年

外推方法:
1. 使用 ΔVth = A × t^n 拟合数据
2. 外推到 10 年 (87,600 小时)
3. 确认 ΔVth_10year ≤ 50mV

示例:
测试数据：ΔVth = 45mV @ 1000h, n = 0.25
外推：ΔVth_10year = 45 × (87600/1000)^0.25 = 45 × 3.06 = 138mV

→ 不满足要求，需要设计优化
```

## 🔍 8. 失效分析流程

### BTI 失效分析步骤

```
失效检测 (ΔVth > 50mV)
    │
    ▼
步骤 1: 电性确认
- 复测 Id-Vg, Id-Vd
- 确认 ΔVth, ΔIdsat, ΔGm
- 确认失效模式 (Vth 漂移为主)
    │
    ▼
步骤 2: 界面态表征
- 电荷泵测量 (Charge Pump)
- 提取界面态密度 Dit
- 确认 Dit 增加
    │
    ▼
步骤 3: 物理分析
- TEM 截面分析
- EELS 元素分析
- 观察 Si/SiO2 界面
    │
    ▼
步骤 4: 根本原因确定
- 界面态密度分布
- 氧化层质量评估
- 工艺缺陷分析
    │
    ▼
步骤 5: 纠正措施
- 氧化工艺优化 (退火温度/时间)
- 氮化处理 (减少 Si-H 键)
- 设计优化 (降低工作电压)
- 重新验证
```

### 常见失效原因

| 失效模式 | 可能原因 | 对策 |
|----------|----------|------|
| ΔVth 过大 | 氧化层质量差 | 优化氧化工艺 |
| n 指数过大 | 界面态密度高 | 氮化处理 |
| 恢复效应弱 | H 原子浓度低 | 形成气体退火 |
| 器件间差异大 | 工艺不均匀 | 工艺优化 |
