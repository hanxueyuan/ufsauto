## 📋 1. 测试目的

评估 MOSFET 在高电场下抗热载流子注入的能力，确保在 10 年使用寿命内不会因 HCI 效应导致器件性能退化。

### HCI 失效机理
```
高电场 (Vds 高) → 载流子获得高能量 (热载流子)
              → 注入栅氧化层
              → Si-H 键断裂，界面态生成
              → 阈值电压漂移、跨导下降
              → 器件性能退化 → 电路失效
```

---

## 🔬 2. 测试原理

### 热载流子产生机制
```
1. 沟道电子在漏端高电场下加速
2. 获得足够能量 (> 3.2 eV for Si/SiO2)
3. 成为"热载流子"
4. 注入栅氧化层
5. 产生界面态 (Dit 增加)
```

### 退化模型
```
ΔVth = A × (Vds)^n × t^m × exp(-Ea/kT)

其中:
- ΔVth = 阈值电压漂移
- Vds = 漏源电压
- n = 电压指数 (典型 3-5)
- m = 时间指数 (典型 0.5-0.7)
- t = 应力时间
```

### 三种应力模式

| 模式 | 偏压条件 | 失效机理 | 适用器件 |
|------|----------|----------|----------|
| **最大衬底电流** | Vg ≈ Vd/2, Vd = Vdd_max | 碰撞电离最强 | NMOS |
| **最大栅电流** | Vg = Vdd_max, Vd = Vdd_max | 热电子注入最多 | PMOS |
| **漏极雪崩** | Vg = 0, Vd = Vdd_max | 寄生 BJT 导通 | LDMOS |

## 📝 5. 测试步骤

### 5.1 初始测量

#### 步骤 1: 转移特性 (Id-Vg)
```
1. Vd = Vdd, Vs = 0, Vb = 0
2. 扫描 Vg: 0 → Vdd
3. 测量 Id
4. 提取 Vth, Gm, S
```

#### 步骤 2: 输出特性 (Id-Vd)
```
1. Vg = 0, 0.5Vdd, Vdd
2. 扫描 Vd: 0 → Vdd
3. 测量 Id
4. 提取 Idsat
```

### 5.2 施加应力

#### 步骤 3: 设置应力条件
```
NMOS (最大衬底电流模式):
- Vd = 1.2 × Vdd_max
- Vg ≈ Vd/2 (调整到最大 Ib)
- Vs = 0
- Vb = 0
- 温度 = 25°C (最坏情况)
```

### 5.3 过程监控

#### 步骤 4: 定时测量
```
时间点：0, 1, 10, 100, 500, 1000h

每个时间点:
1. 暂停应力
2. 冷却至 25°C
3. 测量 Id-Vg, Id-Vd
4. 提取 Vth, Idsat, Gm
5. 计算退化率
6. 判定是否失效
7. 恢复应力
```
