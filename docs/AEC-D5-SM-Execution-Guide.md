## 📋 1. 测试目的

评估金属互连线在机械应力下的抗应力迁移能力，确保在 10 年使用寿命内不会因应力迁移导致开路失效。

### SM 失效机理
```
封装/制造应力 → 金属线内部机械应力
             → 原子扩散 (沿晶界/界面)
             → 空洞形成 (通常在应力集中处)
             → 电阻增加 → 开路 → 器件失效
```

---

## 🔬 2. 测试原理

### 应力来源

| 应力类型 | 来源 | 影响区域 |
|----------|------|----------|
| **热应力** | CTE 不匹配 (Si/金属/钝化层) | 全芯片 |
| **本征应力** | 薄膜沉积过程 | 金属线内部 |
| **封装应力** | 塑封料收缩 | 焊盘下方 |
| **机械应力** | 切割、封装 | 芯片边缘 |

### 空洞形成位置

```
高风险区域:
1. 通孔下方 (Via bottom) - 应力集中
2. 金属线拐角处 - 应力集中
3. 线宽变化处 - 应力梯度
4. 焊盘下方 - 封装应力传递
5. 晶界处 - 原子扩散通道
```

### 应力迁移模型
```
MTTF = A × (1/σ)^n × exp(Ea/kT)

其中:
- MTTF = 平均失效时间
- σ = 机械应力
- n = 应力指数 (典型 2-4)
- Ea = 激活能 (典型 0.5-0.8 eV)
- T = 绝对温度
```

## 📝 5. 测试步骤

### 5.1 测试前准备

#### 步骤 1: Test Key 设计
```
□ 包含不同金属层 (M1 ~ Mn)
□ 包含通孔链结构
□ 包含不同线宽结构
□ 包含长线结构 (> 100μm)
□ 包含拐角结构
□ Kelvin 连接用于精确电阻测量
```

#### 步骤 2: 样品准备
```
□ 选取 77 个样品 (同一批次)
□ 编号：SM-001 ~ SM-077
□ 初始电阻测量 (所有结构)
□ 记录初始数据
□ X-Ray 初始检查 (可选)
```

### 5.2 初始测量

#### 步骤 3: 电阻测量
```
1. 设置探针台温度：25°C
2. 对每个样品测量:
   - M1 线电阻
   - M2 线电阻
   - Via 链电阻
   - 接触电阻
3. 记录所有数据
4. 判定：所有电阻在规格内 → 继续
```

### 5.3 施加应力

#### 步骤 4: 安装样品
```
1. 将样品安装到测试夹具
2. 连接电阻测量线路
3. 放入老化炉
4. 升温至 150°C
```

### 5.4 过程监控

#### 步骤 5: 定时测量
```
测量时间点：0, 10, 50, 100, 200, 500, 1000 小时

每个时间点:
1. 记录温度
2. 测量每个样品的电阻
3. 计算 ΔR/R = (R - R0) / R0 × 100%
4. 判定：ΔR/R > 10% → 失效
5. 记录数据
```

### 5.5 测试结束

#### 步骤 6: 最终测量
```
1. 冷却至 25°C
2. 复测所有电阻
3. 失效样品执行 X-Ray/SAT 分析
4. 记录最终数据
```

## ✅ 7. 判定标准

| 判定 | 标准 |
|------|------|
| **PASS** | 77 个样品中 0 个失效 (ΔR/R ≤ 10% @ 1000h) |
| **FAIL** | ≥1 个样品失效 (ΔR/R > 10%) |

### 寿命外推要求

```
使用条件：125°C
要求：MTTF > 10 年 @ 125°C

外推方法:
1. 使用 Arrhenius 模型拟合
2. 外推到 125°C
3. 确认 MTTF > 87,600 小时
```

## 🔍 8. 失效分析

### 常见失效位置

| 位置 | 风险等级 | 原因 |
|------|----------|------|
| 通孔底部 | 🔴 高 | 应力集中 |
| 金属线拐角 | 🟡 中 | 应力集中 |
| 线宽变化处 | 🟡 中 | 应力梯度 |
| 焊盘下方 | 🟡 中 | 封装应力 |

### 分析流程

```
失效检测 → X-Ray 检查 → SAT 扫描 → 开封 → SEM/EDX → 根本原因
```
