## 📋 1. 测试目的

评估栅氧化层在长期电场应力下的可靠性，确保在 10 年使用寿命内不会发生时变介质击穿 (TDDB) 失效。

### TDDB 失效机理
```
持续电场应力 → 氧化层缺陷累积 → 陷阱电荷增加
            → 泄漏电流增大 → 雪崩击穿
            → 栅极与沟道短路 → 器件失效
```

---

## 🔬 2. 测试原理

### 击穿物理模型
```
阶段 1: 缺陷生成
电场应力 → Si-O 键断裂 → 界面态生成

阶段 2: 陷阱累积
陷阱电荷积累 → 局部电场增强 → 更多缺陷

阶段 3: 渗流路径形成
陷阱连通 → 导电通道形成 → 泄漏电流激增

阶段 4: 热失控
焦耳加热 → 温度升高 → 电流进一步增加 → 击穿
```

### E 模型（电场加速模型）
```
TTF = A × exp(-γ × Eox) × exp(Ea/kT)

其中:
- TTF = 时间到击穿 (Time To Failure)
- A = 材料相关常数
- γ = 电场加速因子 (典型 3-5 cm/MV)
- Eox = 氧化层电场 (MV/cm)
- Ea = 激活能 (典型 0.3-0.6 eV)
- k = 玻尔兹曼常数
- T = 绝对温度
```

### 1/E 模型
```
ln(TTF) = ln(A) + G/Eox - γ × Eox

其中 G 为材料相关参数
```

### 加速因子计算
```
AF = exp[γ × (Estress - Euse)] × exp[(Ea/k) × (1/Tuse - Tstress)]

示例:
- 应力条件：125°C, Eox = 10 MV/cm
- 使用条件：125°C, Eox = 5 MV/cm
- γ = 4 cm/MV

AF = exp[4 × (10 - 5)] = exp(20) ≈ 4.85 × 10⁸

即：10 MV/cm 下测试 1000 小时 ≈ 5 MV/cm 下使用 5.5 亿年
(实际受限于其他失效机理)
```

## 📝 5. 测试步骤

### 5.1 测试前准备

#### 步骤 1: Test Key 设计
```
□ 包含不同氧化层厚度结构 (核心/IO)
□ 包含不同面积栅电容
□ 包含天线比测试结构
□ Kelvin 连接用于精确测量
□ 设计审核通过
```

#### 步骤 2: 样品准备
```
□ 选取 77 个样品 (同一批次)
□ 编号：TDDB-001 ~ TDDB-077
□ 初始电性测试 (C-V, I-V)
□ 记录初始泄漏电流
```

#### 步骤 3: 设备准备
```
□ 老化炉预热至 125°C
□ 校准高压源和皮安表
□ 连接测试夹具
□ 运行空载测试
```

### 5.2 初始测量 (T=0h)

#### 步骤 4: C-V 特性测量
```
1. 设置探针台温度：25°C
2. 对每个样品测量 C-V 曲线
3. 提取氧化层电容 Cox
4. 计算氧化层厚度：Tox = εox × A / Cox
5. 记录数据
```

#### 步骤 5: 初始 I-V 测量
```
1. 扫描栅压：0V → Vstress
2. 测量栅泄漏电流 Ig
3. 记录 Ig @ Vstress
4. 判定：Ig < 100nA → 继续测试
```

### 5.3 施加应力

#### 步骤 6: 安装样品
```
1. 佩戴防静电手环和防热手套
2. 将样品安装到测试夹具
3. 连接高压源和皮安表
4. 放入老化炉
5. 关闭炉门，升温至 125°C
```

#### 步骤 7: 施加栅压
```
1. 温度稳定后 (125°C ± 2°C)
2. 逐步施加栅压至 Vstress (1.2 × Vdd)
3. 监控泄漏电流
4. 记录初始电流 Ig_initial
5. 设置自动监控 (每 10 分钟记录一次)
```

### 5.4 过程监控

#### 步骤 8: 定时测量
```
测量时间点：0, 1, 10, 50, 100, 200, 500, 1000 小时

每个时间点:
1. 记录温度、电压、电流
2. 绘制 Ig-t 曲线
3. 判定：Ig > 1μA → 失效
4. 记录数据
```

### 5.5 测试结束

#### 步骤 9: 最终测量
```
1. 关闭高压源
2. 冷却至 25°C
3. 复测 C-V 和 I-V
4. 记录最终数据
```

| 判定 | 标准 |
