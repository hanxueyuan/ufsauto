# AEC-Q100 D1 电迁移测试 (EM) 执行指导书

**文档编号**: AEC-D1-EM-001  
**版本**: 1.0  
**创建日期**: 2026-03-26  
**依据标准**: AEC-Q100-004, JEDEC JESD611  
**适用产品**: UFS 3.1 车规级存储控制器  
**测试等级**: Grade 1 (-40°C ~ +125°C)

---

## 📋 1. 测试目的

评估金属互连线在大电流密度下的抗电迁移能力，确保在 10 年使用寿命内不会因电迁移导致开路或短路失效。

### 电迁移失效机理
```
大电流密度 → 电子风推动金属原子迁移
          → 空洞 (Void) 或小丘 (Hillock) 形成
          → 导线电阻增加或开路/短路
          → 器件失效
```

---

## 🔬 2. 测试原理

### 物理模型
```
电子流动方向:  ───────────→
金属原子迁移:  ←───────────
              (与电子流相反)

结果:
- 阴极端：原子流失 → 空洞 → 电阻增加 → 开路
- 阳极端：原子堆积 → 小丘 → 可能短路
```

### Black 方程（寿命模型）
```
MTTF = A × (1/J)^n × exp(Ea/kT)

其中:
- MTTF = 平均失效时间 (Mean Time To Failure)
- A = 材料相关常数
- J = 电流密度 (A/cm²)
- n = 电流指数 (典型 1.5-2.0)
- Ea = 激活能 (典型 0.7-1.0 eV)
- k = 玻尔兹曼常数 (8.617×10⁻⁵ eV/K)
- T = 绝对温度 (K)
```

### 加速因子计算
```
AF = (Jstress / Juse)^n × exp[(Ea/k) × (1/Tuse - Tstress)]

示例计算:
- 应力条件：175°C, 1.5 MA/cm²
- 使用条件：125°C, 0.5 MA/cm²
- n = 1.8, Ea = 0.8 eV

AF = (1.5/0.5)^1.8 × exp[(0.8/8.617e-5) × (1/398 - 1/448)]
   = 7.0 × 20.5
   = 143.5

即：175°C 测试 1000 小时 ≈ 125°C 使用 143,500 小时 ≈ 16.4 年
```

## 📝 5. 测试步骤

### 5.1 测试前准备

#### 步骤 1: Test Key 设计验证
```
□ 确认 Test Key 包含所有金属层 (M1, M2, ..., Mn)
□ 确认包含通孔结构 (Via)
□ 确认包含不同线宽 (最小线宽、典型线宽、宽线)
□ 确认 Kelvin 结构用于精确电阻测量
□ 设计文档审核通过
```

#### 步骤 2: 样品准备
```
□ 从合格晶圆中选取 77 个样品 (同一批次)
□ 样品编号：EM-001 ~ EM-077
□ 记录晶圆信息 (批次号、工艺节点、金属层数)
□ 初始电性测试 (所有样品)
□ 存储于干燥柜 (湿度 < 10% RH)
```

#### 步骤 3: 设备准备
```
□ 老化炉预热至设定温度 (175°C)
□ 温度稳定至少 2 小时
□ 校准电流源和电压表
□ 连接测试夹具
□ 运行空载测试确认设备正常
```

### 5.2 初始测量 (T=0h)

#### 步骤 4: 初始电阻测量
```
1. 将样品安装到探针台
2. 设置环境温度：25°C ± 2°C
3. 对每个样品执行以下测量:
   a. 金属线电阻 (R_initial)
   b. 通孔电阻 (R_via_initial)
   c. 接触电阻 (R_contact_initial)
4. 记录所有数据到《EM 测试数据记录表》
5. 判定：所有样品电阻在规格内 → 继续测试
```

**数据记录示例**:
| 样本 ID | R_M1 (Ω) | R_M2 (Ω) | R_Via (Ω) | R_Contact (Ω) | 状态 |
|---------|----------|----------|-----------|---------------|------|
| EM-001 | 125.3 | 98.7 | 0.45 | 0.12 | OK |
| EM-002 | 124.8 | 99.1 | 0.47 | 0.11 | OK |
| ... | ... | ... | ... | ... | ... |

### 5.3 施加应力

#### 步骤 5: 安装样品到老化炉
```
1. 佩戴防静电手环和防热手套
2. 将样品安装到测试夹具
3. 连接电流源和电压表
4. 将夹具放入老化炉
5. 关闭炉门，开始升温
```

#### 步骤 6: 施加电流应力
```
1. 当温度达到 175°C ± 2°C 并稳定后
2. 逐步增加电流至目标值 (1.5 MA/cm²)
3. 监控电压降，计算实际电流密度
4. 记录初始电压 (V_initial)
5. 设置自动监控程序
```

**电流计算示例**:
```
目标电流密度：J = 1.5 MA/cm² = 1.5 × 10⁶ A/cm²
金属线截面积：A = 宽度 × 厚度 = 0.5μm × 0.3μm = 0.15 μm² = 1.5 × 10⁻⁹ cm²

目标电流：I = J × A = 1.5 × 10⁶ × 1.5 × 10⁻⁹ = 2.25 mA
```

### 5.4 过程监控

#### 步骤 7: 定时测量
```
测量时间点：0, 1, 10, 50, 100, 200, 500 小时

每个时间点的操作:
1. 记录当前温度、电流、电压
2. 测量每个样品的电阻
3. 计算电阻变化率：ΔR/R = (R - R_initial) / R_initial × 100%
4. 判定：ΔR/R > 10% → 标记为失效
5. 记录数据到《EM 测试数据记录表》
```

**监控数据记录示例**:
| 时间 (h) | 样本 ID | 温度 (°C) | 电流 (mA) | 电压 (V) | 电阻 (Ω) | ΔR/R (%) | 状态 |
|----------|---------|-----------|-----------|----------|----------|----------|------|
| 0 | EM-001 | 175.2 | 2.25 | 0.282 | 125.3 | 0.0 | OK |
| 100 | EM-001 | 175.1 | 2.25 | 0.285 | 126.7 | 1.1 | OK |
| 200 | EM-001 | 175.0 | 2.25 | 0.291 | 129.3 | 3.2 | OK |
| 500 | EM-001 | 175.2 | 2.25 | 0.310 | 137.8 | 10.0 | FAIL |

#### 步骤 8: 异常处理
```
如发现以下异常，立即记录并报告:
□ 温度超出范围 (> ±2°C)
□ 电流不稳定 (> ±1%)
□ 样品电阻突变 (> 5% 单次变化)
□ 设备故障
□ 任何异常现象

处理流程:
1. 记录异常时间和现象
2. 暂停测试 (如必要)
3. 报告可靠性工程师
4. 评估是否继续测试
5. 记录纠正措施
```

### 5.5 测试结束

#### 步骤 9: 最终测量
```
1. 关闭电流源
2. 老化炉自然冷却至 25°C
3. 取出样品
4. 执行最终电阻测量 (同初始测量)
5. 记录所有数据
```

#### 步骤 10: 失效分析 (如有失效)
```
如有关联失效 (≥1 个样品失效):
1. 拍照记录失效样品
2. 执行无损分析 (X-Ray, SAT)
3. 执行物理分析 (开封, SEM, EDX)
4. 确定失效位置和模式
5. 编写《失效分析报告》
6. 评估是否影响认证
```

## ✅ 7. 判定标准

### Pass/Fail 判定

| 判定 | 标准 |
|------|------|
| **PASS** | 77 个样品中 0 个失效 (电阻变化 ≤ 10%) |
| **FAIL** | 77 个样品中 ≥1 个失效 (电阻变化 > 10%) |

### 寿命外推要求

```
基于测试数据外推使用条件下的 MTTF:

使用条件：125°C, 0.5 MA/cm²
要求：MTTF > 10 年 (87,600 小时)
置信度：90%

计算方法:
1. 使用 Black 方程拟合测试数据
2. 外推到使用条件
3. 计算 90% 置信下限
4. 确认 MTTF_90% > 87,600 小时
```

## 🔍 8. 失效分析流程

### 失效分析步骤

```
失效检测
    │
    ▼
步骤 1: 电性确认
- 复测电阻
- I-V 特性曲线
- 确认失效模式 (开路/短路/电阻增加)
    │
    ▼
步骤 2: 无损分析
- X-Ray 检查 (空洞、小丘)
- SAT 超声扫描 (分层)
- 光学显微镜 (表面形貌)
    │
    ▼
步骤 3: 物理分析
- 开封 (去钝化层)
- SEM (扫描电镜)
- EDX (能谱分析)
- FIB (聚焦离子束) 切片
    │
    ▼
步骤 4: 根本原因确定
- 失效位置定位
- 失效模式确认
- 失效机理分析
    │
    ▼
步骤 5: 纠正措施
- 设计改进 (加宽金属线、增加通孔)
- 工艺改进 (退火、金属成分优化)
- 重新验证
```

### 常见失效模式

| 失效模式 | 特征 | 原因 | 对策 |
|----------|------|------|------|
| **空洞导致开路** | 电阻无限大 | 阴极端原子流失 | 增加金属线宽 |
| **小丘导致短路** | 电阻接近 0 | 阳极端原子堆积 | 优化金属晶粒结构 |
| **电阻渐变增加** | 电阻缓慢上升 | 空洞逐渐形成 | 降低电流密度 |
| **通孔失效** | 通孔电阻增加 | 通孔底部空洞 | 增加通孔数量 |

## ⚠️ 9. 安全注意事项

### 高温安全
```
□ 佩戴防热手套操作高温样品
□ 使用专用工具取放样品
□ 避免皮肤直接接触高温表面
□ 老化炉周围设置警示标识
□ 熟悉紧急停机程序
```

### 电气安全
```
□ 确保设备接地良好
□ 高压操作时双人作业
□ 佩戴绝缘手套
□ 测试前检查线路连接
□ 禁止带电插拔
```

### 静电防护
```
□ 佩戴防静电手环
□ 使用防静电工作台
□ 样品存储在防静电容器中
□ 避免直接触摸芯片引脚
```

### 应急处理
```
烫伤:
- 立即用冷水冲洗 15 分钟
- 不要涂抹药膏
- 严重者立即就医

电击:
- 立即切断电源
- 不要直接接触触电者
- 呼叫急救
```
