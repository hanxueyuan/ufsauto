## 二、详细测试条件

### 2.1 HTOL（高温工作寿命）

**测试目的**：验证芯片在高温和电压应力下的长期可靠性

**测试条件**：
| 参数 | 条件 | 备注 |
|------|------|------|
| 温度 | 125℃（Grade 1）或 150℃（Grade 0） | 根据产品等级选择 |
| 电压 | 根据 device 确定（通常 1.05×Vcc ~ 1.1×Vcc） | 不是固定 1.1 倍，需根据 absolute max rating |
| 时间 | 1000 小时（标准）或根据 AF 计算 | 可根据 life time 需求调整 |
| 样本量 | 77 颗（LTPD=1, 60% CL）或 231 颗（LTPD=1, 90% CL） | 根据置信度要求 |
| 测试模式 | 全功能运行（worst case） | 不建议用 DFT pattern |

**判定标准**：
- 测试前后电性参数变化 < 10%
- 无功能性失效
- IV 曲线变化 < 30%

**加速因子计算**：
```
温度加速因子：AFT = exp[(Ea/k) × (1/Tu - 1/TA)]
  - Ea = 0.7 eV（典型值）
  - k = 8.617×10^-5 eV/K
  - Tu = 使用温度（K）
  - TA = 加速温度（K）

电压加速因子：AFV = exp[γ × (VA - VU)]
  - γ 由 Fab 提供（典型 1-3）

总加速因子：AF = AFT × AFV
等效寿命 = 测试时间 × AF
```

### 2.2 HAST / uHAST（高加速应力测试）

**测试目的**：验证封装在温湿度应力下的可靠性

**测试条件**：
| 参数 | HAST（饱和） | uHAST（非饱和） |
|------|-------------|----------------|
| 温度 | 130℃ | 110℃ / 130℃ |
| 湿度 | 85% RH | 85% RH |
| 压力 | 2.7 atm（绝对压力） | 常压 |
| 偏压 | 80% 额定电压 | 80% 额定电压 |
| 时间 | 96 小时 | 96 小时 |
| 样本量 | 77 颗 | 77 颗 |

**判定标准**：
- 无分层、开裂
- 电性参数变化 < 10%
- SAM 检查无异常

---

### 2.3 TC / TCT（温度循环 / 温度冲击）

**测试目的**：验证封装在温度交变应力下的可靠性

**测试条件**：
| 参数 | TC（温度循环） | TCT（温度冲击） |
|------|---------------|----------------|
| 高温 | 125℃ | 150℃ |
| 低温 | -65℃ | -65℃ |
| 保温时间 | 15 分钟 | 15 分钟 |
| 转换时间 | < 5 分钟 | < 20 秒 |
| 循环数 | 1000 cycles | 1000 cycles |
| 样本量 | 77 颗 | 77 颗 |

**判定标准**：
- 无开裂、分层
- 电性参数变化 < 10%
- X-ray/SAM 无异常

---

### 2.4 HTSL（高温存储）

**测试目的**：验证芯片在高温存储条件下的可靠性

**测试条件**：
| 参数 | 条件 |
|------|------|
| 温度 | 150℃（Grade 1）或 175℃（Grade 0） |
| 时间 | 1000 小时 |
| 样本量 | 77 颗 |
| 偏压 | 无偏压 |

**判定标准**：
- 电性参数变化 < 10%
- 无外观异常

---

### 2.5 Precon（预处理）

**测试目的**：模拟 SMT 回流焊前的吸湿和热应力

**测试条件**：
| 步骤 | 条件 | 时间 |
|------|------|------|
| 1. 烘烤 | 125℃ | 24 小时 |
| 2. 吸湿 | 30℃/60% RH | 192 小时（MSL3） |
| 3. 回流焊 | 260℃峰值 | 3 次 |
| 4. 测试 | 电性测试 + SAM | - |

**判定标准**：
- 无爆米花效应
- SAM 检查无分层
- 电性参数正常

---

### 2.6 ESD（静电放电）

**测试目的**：验证芯片抗静电放电能力

**测试条件**：
| 测试项 | 标准 | 最低要求 | 推荐要求 |
|--------|------|----------|----------|
| HBM | AEC-Q100 A114 | 2000V | 4000V |
| CDM | AEC-Q100 A114 | 500V | 1000V |
| MM | AEC-Q100 A114 | 200V | 可选 |

**判定标准**：
- 测试后功能正常
- IV 曲线变化 < 30%
- 无 latch-up

---

### 2.7 Latch-up（闩锁效应）

**测试目的**：验证芯片抗闩锁效应能力

**测试条件**：
| 参数 | 条件 |
|------|------|
| 测试温度 | 室温 + 高温（85℃） |
| 过压 | VDD + 10% 或 VDD + 0.5V（取大者） |
| 过流 | 100mA（I/O），200mA（电源） |
| 判定 | 无 latch-up 触发 |

---

### 2.8 耐久性 / 数据保持（Endurance / Retention）

**测试目的**：验证闪存单元的擦写寿命和数据保持能力

**测试条件**：
| 参数 | 条件 |
|------|------|
| 耐久性 | 10K cycles（TLC）或按规格 |
| 数据保持 | 10 年 @ 25℃（或等效加速） |
| 温度 | 85℃（加速） |
| 样本量 | 77 颗 |

**判定标准**：
- 耐久性后 ECC 可纠正
- 数据保持后 UBER < 10^-15

## 四、失效判定与 FA 流程

### 4.1 失效判定标准

| 失效类型 | 判定标准 |
|----------|----------|
| 功能失效 | 无法通过功能测试 |
| 参数失效 | 电性参数超出规格 |
| 潜在失效 | IV 曲线变化 > 30% |
| 物理失效 | 开裂、分层、腐蚀等 |

### 4.2 FA 流程

```
1. 失效确认
   └─ 复测验证
   └─ 记录失效模式

2. 非破坏性分析
   └─ X-ray 检查
   └─ SAT（声学显微镜）
   └─ IV 曲线测试
   └─ EMMI（微光显微镜）

3. 破坏性分析
   └─ 开盖（Decap）
   └─ 去胶（De-lid）
   └─ SEM/EDX
   └─ FIB 切片

4. 根因定位
   └─ 失效机理分析
   └─ 改进建议

5. 报告输出
   └─ 8D 报告
   └─ CAPA（纠正预防措施）
```

## 六、注意事项

1. **HTOL 电压应力**：根据 device 确定，不是固定 1.1 倍 Vcc
2. **样本量**：根据 LTPD 和置信度要求选择
3. **测试模式**：HTOL 建议使用全功能 worst case pattern，不是 DFT
4. **失效分析**：IV 曲线变化>30% 判定为 ESD 损伤
5. **批次管理**：变更管理严格按照 AEC-Q100 要求
