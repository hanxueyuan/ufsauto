## 🎨 颜色效果说明

### 终端颜色代码

| ANSI 代码 | 颜色 | 用途 |
|-----------|------|------|
| `\033[30m` | 黑色 | - |
| `\033[31m` | 红色 | ERROR, FAIL |
| `\033[32m` | 绿色 | PASS, INFO, 成功 |
| `\033[33m` | 黄色 | WARNING |
| `\033[34m` | 蓝色 | RESULT, 性能结果 |
| `\033[35m` | 紫红色 | FAIL (测试失败) |
| `\033[36m` | 青色 | DEBUG |
| `\033[31;1m` | 红色加粗 | CRITICAL |
| `\033[0m` | 重置 | 恢复默认颜色 |

### 视觉效果

- **绿色** (✅): 传递积极反馈，一眼看到成功
- **红色** (❌): 立即吸引注意力到问题
- **黄色** (⚠️): 温和警示，不中断流程
- **蓝色** (📊): 中性信息，适合数据和结果
- **青色** (🔍): 调试信息，不干扰主流程
