# 车规UFS 3.1产品需求规格书 V1.0

## 1. 范围
本规格书定义了符合车载应用要求的UFS（通用闪存存储）3.1产品的功能、性能、可靠性、安全性及环境适应性要求，适用于车载信息娱乐系统、高级驾驶辅助系统（ADAS）、自动驾驶系统、数字仪表盘等车载存储应用场景。

## 3. 术语和定义
### 3.1 UFS
通用闪存存储（Universal Flash Storage），是一种设计用于消费电子和汽车应用的高性能串行接口存储标准。
### 3.2 AEC-Q100
汽车电子委员会（AEC）制定的用于集成电路的应力测试认证标准，确保元器件适用于汽车环境。
### 3.3 ISO 26262
道路车辆功能安全标准，定义了汽车电子电气系统的功能安全要求和开发流程。
### 3.3 ASIL
汽车安全完整性等级（Automotive Safety Integrity Level），ISO 26262定义的风险分类系统，从A到D等级，D级安全要求最高。
### 3.4 TBW
总写入字节数（Terabytes Written），衡量固态硬盘寿命的指标。
### 3.5 DWPD
每日全盘写入次数（Drive Writes Per Day），衡量固态硬盘在质保期内每天可写入的次数。

## 4. 产品概述
### 4.1 产品定位
本产品是符合UFS 3.1标准的车规级存储芯片，专为严苛的车载环境设计，满足高性能、高可靠性、高安全性的存储需求，支持-40℃~105℃的宽温工作范围，符合AEC-Q100 Grade 2认证要求，支持ISO 26262 ASIL B功能安全等级，可应用于各类车载电子系统。
### 4.2 应用场景

- 行车记录仪（DVR）

- 智能座舱域控制器

## 5. 功能规格要求
### 5.1 接口特性
- 符合JEDEC JESD220E UFS 3.1标准
- M-PHY V4.1，HS-G4，每通道速率11.6Gbps
- 支持1或2通道配置（默认2通道）
- UniPro V1.8协议层
- 支持命令队列（Command Queue），最大队列深度32
- 支持HS-Gear4自适应调节
- 支持低功耗模式（Hibern8、Sleep、Stall）
### 5.2 命令集支持
- 支持所有UFS 3.1标准命令
- 支持安全擦除（Secure Erase）、快速安全擦除（Crypto Erase）
- 支持TRIM/UNMAP命令
- 支持厂商自定义命令，包括健康状态查询、坏块信息查询、温度监控等
### 5.3 高级功能
#### 5.3.1 写入 booster（Write Booster）
- 支持UFS 3.1写入增强功能，提升随机写入性能
- 动态SLC缓存配置，根据工作负载自动调整缓存大小
- 最低保证写入booster容量不小于标称容量的5%
#### 5.3.2 主机性能提升器（Host Performance Booster, HPB）
- 支持HPB 1.0功能，减少L2P映射表读取开销
- 支持HPB区域大小可配置，最大支持64MB映射区域
#### 5.3.3 上下文识别（Context ID）
- 支持最多8个上下文ID，主机可根据应用场景标记不同优先级的IO请求
- 支持基于上下文ID的服务质量（QoS）调度
#### 5.3.4 端到端数据保护
- 支持端到端数据保护（End-to-End Data Protection）
- 支持16位CRC校验，检测数据传输过程中的错误
- 支持数据重传机制，提升传输可靠性
#### 5.3.5 电源管理
- 支持多个电源档位，根据工作负载动态调节功耗
- 支持主动电源管理（Active Power Management）
- 支持系统睡眠唤醒时的快速上电响应
### 5.4 安全功能
#### 5.4.1 硬件加密
- 支持AES-256硬件加密引擎
- 支持XTS加密模式
- 支持全盘加密（FDE）和分区加密
- 加密性能不低于原生性能的95%
#### 5.4.2 安全启动
- 支持安全启动功能，验证固件完整性
- 支持固件签名，防止恶意固件篡改
- 支持安全固件更新，更新过程中掉电不损坏设备
#### 5.4.3 密钥管理
- 支持最多8个独立加密密钥
- 支持密钥安全存储，密钥不可被外部读取
- 支持密钥销毁功能，快速安全擦除所有数据
#### 5.4.4  replay protected memory block（RPMB）
- 支持RPMB分区，容量不小于16MB
- 支持RPMB分区的安全读写操作
- 符合JEDEC JESD220E标准中RPMB的所有要求

## 6. 性能指标要求
### 6.1 顺序读写性能
| 容量 | 顺序读取（MB/s） | 顺序写入（MB/s） |
|------|------------------|------------------|
| 64GB | ≥1800 | ≥700 |
| 128GB | ≥2100 | ≥1000 |
| 256GB | ≥2100 | ≥1400 |
| 512GB | ≥2100 | ≥1800 |
| 1TB | ≥2100 | ≥2000 |
*注：以上为写入booster开启时的性能指标*
### 6.2 随机读写性能
| 容量 | 随机读取（IOPS, 4K） | 随机写入（IOPS, 4K） |
|------|----------------------|----------------------|
| 64GB | ≥400,000 | ≥60,000 |
| 128GB | ≥420,000 | ≥80,000 |
| 256GB | ≥420,000 | ≥100,000 |
| 512GB | ≥420,000 | ≥150,000 |
| 1TB | ≥420,000 | ≥200,000 |
*注：以上为写入booster开启时的性能指标*
### 6.3 性能一致性
- 稳态随机写入性能不低于峰值性能的70%
- 长时间运行下性能波动不超过10%
- 温度变化对性能的影响不超过15%
### 6.4 响应时间
- 平均读取响应时间 ≤ 0.1ms
- 平均写入响应时间 ≤ 0.3ms
- 最大读取响应时间 ≤ 10ms
- 最大写入响应时间 ≤ 50ms
### 6.5 功耗特性
- 工作功耗 ≤ 1.8W（满载）
- 活动待机功耗 ≤ 80mW
- 休眠功耗 ≤ 1mW
- 深度休眠功耗 ≤ 100μW

## 7. 可靠性要求
### 7.1 使用寿命
| 容量 | TBW（总写入字节数） | DWPD（5年质保） |
|------|----------------------|------------------|
| 64GB | ≥300 | ≥0.25 |
| 128GB | ≥600 | ≥0.25 |
| 256GB | ≥1200 | ≥0.25 |
| 512GB | ≥2400 | ≥0.25 |
| 1TB | ≥4800 | ≥0.25 |
*注：以上数值为JEDEC JESD218标准下的标称值*
### 7.2 平均无故障时间（MTBF）
- MTBF ≥ 3,000,000小时（@25℃工作温度）
### 7.3 坏块管理
- 初始坏块率 ≤ 0.1%
- 支持动态坏块管理，自动替换运行中产生的坏块
- 保留块数量不低于总块数的2%
### 7.4 掉电保护
- 支持异常掉电保护，掉电时保证正在写入的数据不丢失
- 支持元数据保护，掉电后元数据不损坏
- 掉电恢复时间 ≤ 100ms
### 7.5 ECC纠错能力
- 支持LDPC纠错算法
- 最大可纠正每1KB数据中的40bit随机错误
- 支持多级纠错，提升NAND闪存可靠性

## 8. 车规认证要求
### 8.1 AEC-Q100认证
- 符合AEC-Q100 Grade 2认证要求
- 工作温度范围：-40℃ ~ 105℃（环境温度）
- 存储温度范围：-55℃ ~ 125℃
#### 8.1.1 环境应力测试
| 测试项目 | 要求 | 标准 |
|----------|------|------|
| 温度循环（Temperature Cycle） | -40℃~105℃，1000次循环 | AEC-Q100-005 |
| 高温存储（High Temperature Storage Life） | 125℃，1000小时 | AEC-Q100-003 |
| 低温存储（Low Temperature Storage Life） | -55℃，1000小时 | AEC-Q100-003 |
| 高温高湿（Humidity Bias） | 85℃/85%RH，1000小时 | AEC-Q100-002 |
| 高温工作寿命（High Temperature Operating Life） | 105℃，1000小时 | AEC-Q100-004 |
| 温度冲击（Temperature Shock） | -40℃~105℃，100次循环 | AEC-Q100-006 |
#### 8.1.2 机械应力测试
| 测试项目 | 要求 | 标准 |
|----------|------|------|
| 机械冲击（Mechanical Shock） | 1500G，0.5ms，半正弦波 | AEC-Q100-007 |
| 振动（Vibration） | 20~2000Hz，20G，随机振动 | AEC-Q100-008 |
| 可焊性（Solderability） | 符合J-STD-002要求 | AEC-Q100-009 |
| 耐焊接热（Resistance to Soldering Heat） | 260℃，10秒 | AEC-Q100-007 |
### 8.2 EMC/EMI要求
- 符合GB/T 37376车载电子电磁兼容性要求
- 辐射骚扰（RE）符合CISPR 25 Class 3要求
- 传导骚扰（CE）符合CISPR 25 Class 3要求
- 静电放电（ESD）：接触放电±8kV，空气放电±15kV，符合ISO 10605标准
- 电快速瞬变脉冲群（EFT）：±4kV，符合ISO 7637-2标准
- 浪涌抗扰度：±2kV，符合ISO 7637-2标准
- 电源电压瞬时变化抗扰度符合ISO 16750-2标准要求
### 8.3 IATF 16949质量管理体系
- 产品开发和生产流程符合IATF 16949:2016标准要求
- 提供PPAP（生产件批准程序）文件包
- 支持APQP（产品质量先期策划）流程
- 可提供完整的FMEA（失效模式与影响分析）报告
- 提供全生命周期的产品变更管理（PCN）服务，变更提前12个月通知

## 9. 功能安全要求（ISO 26262）
### 9.1 安全目标
- 防止因存储数据损坏导致的车辆危险行为
- 防止因存储设备故障导致的车辆功能失效
- 确保数据存储的完整性和可靠性
### 9.2 ASIL等级
- 产品开发流程符合ISO 26262:2018 ASIL B等级要求
- 支持最高ASIL D级系统的集成应用
### 9.3 安全机制
#### 9.3.1 数据完整性保护
- 端到端CRC校验，覆盖从主机接口到NAND存储的全路径
- 元数据冗余存储，支持多副本校验
- 数据读回验证机制，写入后自动验证数据正确性
#### 9.3.2 错误检测与纠正
- 支持ECC纠错，可纠正单比特和多比特错误
- 支持不可纠正错误（UECC）检测和上报
- 支持存储单元健康监控，提前预警潜在故障
#### 9.3.3 故障诊断与上报
- 内置丰富的诊断功能，可检测和上报多种故障模式
- 支持标准UFS健康状态查询接口
- 支持故障日志记录，可追溯最近100次异常事件
#### 9.3.4 安全状态控制
- 检测到严重故障时，可进入安全状态，避免故障扩散
- 支持主机控制的安全复位功能
- 支持故障隔离，局部故障不影响整体功能
### 9.4 安全文档
- 提供符合ISO 26262要求的安全手册（Safety Manual）
- 提供FMEA报告
- 提供FMEDA（失效模式影响和诊断分析）报告
- 提供安全案例（Safety Case）
- 提供独立第三方的ASIL认证证书
